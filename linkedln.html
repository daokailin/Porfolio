Linkedln ios Redesign

DISCOVER VARIOUS PROFESSIONAL MEETING OPPORTUNITIES

Project Introduction

The objective of this project is to redesign the linkedln app's group feature that will increase users' awareness of off-line meetings, professional networking sessions, and resources while ensuring this piece of information is easily accessible and user-friendly. Linkedln users should have more opportunities to meet with local professionals and engage in networking.

Identify the Problems

The current Linkedln mobile app is lacking the basic meetup function as a social media platform geared to professionals. The existing "My Network" function doesn't provide users with appropriate meeting opportunities to communicate with people who share the same interests and passions in person.

Thus, we are dedicated to implement a new function that helps users to learn knowledgeable skills and industry experience through "real connection."

User Research Methods

1. We sent out online google form surveys to our friends and colleagues who have used the linkedln app before. The goal of using this method was to see how college graduates interact with the existing group feature and find out the trends of their expectations or needs of networking experience.

2. Our team conducted two observational interviews to collect more in-depth information about these participants' user experiences. We asked them to share their experience of meeting with professionals on linkedln, and we recorded how they used the My Network feature to find their interested groups.

Jinghao is interviewing experienced linkedln users.

Competitor Analysis

Since group feature has been widely used in other popular social media platforms, we used Competitor analysis to research how well are they served the users and what's their primary features.

As we can see, most competitive products are lack of social elements which helps users to gain the sense of communities and belongings. Without the function that enables users to connect with others personally, the group feature will lose its value. Additionally, we found that we could add a filter to allow users to target their interested groups/events, and we are also considering implement recommendation system to remind users the upcoming events or interesting news.

Based on the data we collected from user testings, 80% of the interviewees stated that they would show up to an event because of the group they've joined. In light of this, we decided to implement a group feature to help users to discover their interested communities.

Redesign Screens

Home Page (old users)

This is the home page. Users have 2 points of entry for the group funtion. In addition to the group function button at the bottom bar, user can also access it through tooltips on the top of the screen, which is more visible and reminds them of this function. Based on user feedback, we intended to make the experience of accessing the funciton differently based on user’s habit and frequency. So ideally the top tooltips can be toggled off automatically and the app knows that the user has been accessing the function frequently. It will appear again if the users haven’t used it in a while.

Home Page (new users)

If a new user hasn’t joined any groups yet, he/she will see this “new user” version screen once he enters the group function through either of the points of entry. There is a “dicover groups” section in the middle of the screen, and users can manually search groups. This section exists to invite the users to explore the function and gives them a starting point, while also telling them what this function does. Below that are various recommended groups that are based on users’ career interests and profile information.

Group List

We created several variants for the group list screen. With these alternatives, we tried to explore different ways of presenting event details.

70% of the users stated that they liked the calendar view since it's a direct visual representation of the event timeline. We simplified the calendar from monthly to weekly in order to retain only the necessary information and conserve space. Different button colors indicate the event status, and the small icon right next to the event title allows users to import these linkedln events to their favorite calendar apps.

Group Detailed Page

This screen shows the detailed information of a specific group based in San Diego. Users can view the information about this group and its members. By tapping the area where it says “Start Conversation”, user can post messages or share pictures and videos. This function is available only after the users has joined the group. Furthermore, he/she can view the upcoming group events and other people’s posts.

Event Detailed Page

This is the page where it shows all the information of a specific event. The biggest feature is that users can easily connect with the experienced professionals who host the event by clicking the “hosted by” area. For attendees, names of those whom the user has a conection with on LinkedIn will be displayed first, which increases their motivation to attend. We believe this change can be a subtle way to increase user’s desire to go to events, by meeting with his friend first, then also other people. This progression for meeting new people is also what we learned from user feedback: you build stronger connections if you and someone has attended few events together. Through such cycle, a user’s connections become more diverse and stronger, which achieves our redesign goals.

Reflection

This is the first redesign project I've done, but I still found a lot of interesting things during the design process. The first challenging part was that we had to find out what resources are available to us to re-create the style and layout of the current linkedln app. We were struggled to search for the fonts and other UI elements. Another surprising thing was that we always found redundant features or elements after we finalized an idea, we had to spend more time on creating a brand new feature.

Final Thoughts and Future Plan

Reflecting on our design process, the most exciting feature we developed was the event function that helps different group members to network in professional setting. The data we collected signified that users would like to have various kinds of events other than professional meetings.

For our future plan, we want to build more features and have deep integration with more of the connection tools. Additionally, we want to enlarge our user groups and promote this product to working professionals instead of only focusing on college graduates.
